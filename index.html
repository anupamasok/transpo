The Transportation Problem

The Transportation problem is a special type of linear programming problem where the objective is to minimize the cost of distributing a product from a number of sources or origins to a number of destinations.

The Northwest Corner Method (or upper left-hand corner) is a heuristic that is applied to the Transportation Model, which ensures that there is an initial basic feasible solution.Other methods for obtaining an initial basic solution are the Minimum Cell Cost Method and the Vogel Approximation Model. Among these the Northwest Corner Method involves the least number of calculations. Read more
